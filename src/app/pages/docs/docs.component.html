¿How to Run?

  1. First of one. Sign up to LemonBooster Register and then Login

  2. Go to DigitalOcean, Linode or run a VPS with your favourite provider.

    • A VPS of 2gb ram and 50gb storage is recommended.

  3. On your Shell → ssh root@[YourVpsIP]

  4. Once inside VPS → mkdir ~/tools

  5. Go to the tools folder → cd ~/tools

  6. Clone the repo → git clone https://github.com/bonino97/LemonBooster-v2.git

  7. Go to Config LemonBooster folder → cd ~/tools/LemonBooster-v2/Config-LemonBooster-v2

  8. Give privileges to the SH Installer. → chmod +x Install-LemonBooster-Ubuntu20.04.sh

    • If you have a VPS with Ubuntu 18.04. → chmod +x Install-LemonBooster-Ubuntu18.04.sh

  9. Run the Installer → ./Install-LemonBooster-Ubuntu20.04.sh

    • The script gonna install the database for save results, all the tools used by LemonBooster and all the requireds dependencies.

    • The installer leave your VPS ready to configure on the WEB.

  10. Once the Installer finish. Go to Settings. and load the VPS Ip .

  11. If everything installed correctly, you should see [Status: ]. In the left Sidebar options.

  12. IMPORTANT! → If you want to see the proccess running:

    1. Go to Config Folder → cd ~/tools/LemonBooster-v2/Config-LemonBooster-v2

    2. Give privileges to the ViewExecution SH. → chmod +x ViewExecution.sh

    3. Run SH. → ./ViewExecution.sh

    4. By this way, you can execute the tools, and see all the proccess running (tools included).

Start Server before reboot VPS

  1. Go to Config folder → cd ~/tools/LemonBooster-v2/Config-LemonBooster-v2

  2. Give privileges to StartServerBeforeReset sh → chmod +x StartServerBeforeReset.sh

    • If you don't see any mentioned file. Don't remember to follow Update Server

  3. Execute → ./StartServerBeforeReset.sh

Update Server

  1. Go to LemonBooster folder → cd ~/tools/LemonBooster-v2

  2. Reset local changes → git reset --hard

  3. Pull new changes → git pull

¿How to add your own Wordlist?

  • Copy and Paste the List inside the Lists folder → cp Example.txt ~/tools/LemonBooster-v2/API-LemonBooster-v2/Common/Lists

Features:

  1. OneClick complete Scanning.[Run Subdomain Enumeration, Alives, Screenshots, Spidering]

  2. Subdomain Enumerations. [Findomain, Subfinder, Assetfinder, Amass, Github Search, Permutations]

  3. Alives. [Httprobe]

  4. Subdomain Response Codes. [Hakcheckurl]

  5. Screenshots. [Aquatone]

  6. Js Scanning. [Linkfinder]

  7. Waybackurls. [Waybackurls]

  8. Spidering. [GoSpider, Hakrawler].

  9. Dir Bruteforcing. [Dirsearch].

  10. Get multiples loaded lists to use.

  11. Save results on MongoDB.

  12. Compare Old Results with New Results in each Scanning.

  13. New Results in Monitoring Option.
